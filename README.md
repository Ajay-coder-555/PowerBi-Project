# PowerBi-Project
📊 ShopNest Business Performance Dashboard

📌 Project Overview

This Power BI project analyzes the business performance of an e-commerce store using interactive dashboards and visualizations.

🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Excel

📈 Dashboard Features

- Total Revenue Analysis
- Total Orders Analysis
- State-wise Sales Analysis
- Product Category Analysis
- Payment Type Analysis
- Monthly & Yearly Revenue Trends
- Delayed Orders Analysis
